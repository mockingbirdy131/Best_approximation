# Best_approximation

Данный проект реализует построение наилучшего приближения непрерывной функции из класса на отрезке с использованием метода конечных элементов и базисных функций узлов. Приближение строится на основе значений функции в случайных точках внутри каждого конечного элемента путём решения системы линейных алгебраических уравнений.

В программе реализуются следующие шаги:

1. Задание параметров:
   - Концы отрезка [a, b]
   - Количество конечных элементов K
   - Количество узлов N на конечном элементе
   - Количество случайных внутренних точек L на конечном элементе
   - Количество точек визуализации M_vis

2. Построение сетки:
   - Вычисление общего количества узлов M
   - Равномерная сетка с шагом h между узлами

3. Генерация случайных точек:
   - L > N внутренних случайных точек на каждом конечном элементе
   - Равномерное распределение случайных чисел

4. Построение наилучшего приближения:
   - Базисные функции — интерполяционные многочлены Лагранжа
   - Формирование СЛАУ для нахождения коэффициентов приближения -- матрица Грама имеет ленточную структуру (блоки на диагонали)
   - Использование функции из библиотеки "Eigen" для решения СЛАУ (методом LU-разложения) -- библиотеку необходимо установить перед компиляцией

5. Анализ погрешности:
   - Вычисление абсолютной и относительной погрешности в случайных точках
   - Вычисление абсолютной и относительной погрешности на сетке (шаг h/100)
   - Использование трех различных норм векторов (сумма модулей, корень из суммы квадратов модулей, максимум модуля)

6. Визуализация:
   - Построение совмещённого графика функции (сплошная линия) и наилучшего приближения (пунктир)
   - Выделение внутренних узлов конечных элементов одним цветом, концов - другим
